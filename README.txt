This Readme contains a little help for the Program "polymer_elasticity"

To start the Program:
	- open the Script "polymer_elasticity" in the Matlab-Editor and press "F5" or type "polymer_elasticity" within the Command Window of the Matlab-Editor
	
To obtain a more detailed Help:
	- Within the running Script, hit the Menu Point "Polymer Elasticity" in the Figures Menu and go to "Help"
	- Open the Help.docx-File in the Help-Folder of the polymer_elasticity Folder
